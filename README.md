# 📱 Task Manager App

## 📌 Description

This is an Android application developed as part of an assignment. The app allows users to manage their daily tasks with features like login, task list, adding tasks, and more.

---

## 🚀 Features

* Login Screen using Activity
* Dashboard with RecyclerView
* Add Task using Dialog
* Fetch Tasks from API (Retrofit)
* Save User Data using DataStore
* Background Task using WorkManager
* WebView integration

---

## 🛠 Technologies Used

* Kotlin
* Android Studio
* RecyclerView
* Retrofit
* DataStore
* WorkManager
* WebView

---

## ▶️ How to Run

1. Open project in Android Studio
2. Connect emulator/device
3. Click Run ▶️
